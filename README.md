# Silent Financial Stress Detection Dashboard

An analytics-driven dashboard that identifies early signs of financial stress using bank transaction behavior.

## Features
- Portfolio-level financial stress monitoring
- Customer-level stress analysis
- Top-risk customer prioritization
- Explainable, rule-based stress scoring
- Business validation using churn insights

## Tech Stack
- Python, Pandas
- Streamlit
- Matplotlib
